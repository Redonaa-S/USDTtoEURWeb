# USDTtoEURWeb

USDTtoEURWeb is a lightweight web application that converts **USDT (Tether)** to **EUR** using the current exchange value. The tool provides a fast and simple way to estimate the value of cryptocurrency in euros.
Live Demo
You can access the application here:
https://usdt-to-eurw-eb.vercel.app/

# Features

* Fast USDT to EUR conversion
* Simple and user-friendly interface
* Accessible from any device with a browser
* Designed for quick personal cryptocurrency calculations

## Tech Stack

* React
* Vite
* JavaScript
* Deployed with Vercel

# Deployment

The application is deployed using Vercel, allowing the project to be accessed online from anywhere without installation.

## Purpose

This project was created as a practical web tool and as part of a learning portfolio for web development.
